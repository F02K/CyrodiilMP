# CyrodiilMP Inspector Dashboard

The dashboard is a local web UI for browsing full research runs and starting a new metadata collection pass.

## Run

Foreground:

```powershell
.\scripts\run-dashboard.cmd -Port 5088
```

Background:

```powershell
.\scripts\start-dashboard-background.cmd -Port 5088
```

Stop background process:

```powershell
.\scripts\stop-dashboard.cmd
```

Then open:

```text
http://127.0.0.1:5088
```

## Features

- Shows the configured game path from `game-path.txt`.
- Lists generated full-research runs.
- Displays `report.md` in a readable view.
- Displays CSV outputs as tables.
- Starts a new full research pass from the browser.
- Shows live job status and recent job logs.

The dashboard reads metadata generated by the research scripts. It does not copy, serve, or extract proprietary game assets.
