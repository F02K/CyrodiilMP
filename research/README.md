# Research Workspace

This folder is for evidence, dumps, notes, and generated reports used while building CyrodiilMP.

Keep:

- focused notes in `runs/`
- small hand-written summaries that explain a decision
- references to dump folders that led to a real code change

Do not commit generated bulk output unless it is intentionally preserved:

- runtime dump CSVs
- full research reports
- FModel indexes
- quick-scan inventories
- dashboard runtime state
- smoke-test logs

Most generated research output is ignored by `.gitignore`.
